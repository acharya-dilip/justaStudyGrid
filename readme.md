# JustaGridSweeper
JustaGridSweeper is a Minesweeper Game made completely in C<br>
Get those mines sweeped off the grid! OH NOO! EXPLOSION!! BOOOM!!!<br>
So get on sweeping the grid made completely using gtk and C.

# How to user?
- Open the game 
- Start Clickin
- Be Ware of the bombs
- Sweep the grid though I know you won't
- But keep trying until you do anyways
- Once you die restart screen will popup
- Click Restart and jump right back in

# What works?
- The logic for random mines
- The proximal numbers around the mines 
- Clicking sound for the grid
- Bomb explosion
- A winning system
